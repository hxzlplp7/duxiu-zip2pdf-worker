# 读秀ZIP转PDF工具 - 使用说明

## 📖 这是什么？

这是一个可以把读秀下载的ZIP文件转换成PDF的在线工具。不需要自己的服务器，完全基于Cloudflare云服务。

## 🎯 主要功能

✨ **上传ZIP文件** - 支持读秀的ZIP/UVZ/CBZ格式  
⚡ **自动转换** - 自动解压、识别图片、生成PDF  
📥 **直接下载** - 转换完成后直接下载PDF  
☁️ **云端存储** - 使用Cloudflare R2存储，安全可靠  
💎 **精美界面** - 现代化设计，操作简单

## 🚀 如何使用？

### 方式一：直接使用（推荐新手）

如果你只想使用，不想自己部署：

1. 等待部署完成后，会得到一个网址
2. 打开网址，上传ZIP文件
3. 点击"开始转换"
4. 等待完成后下载PDF

### 方式二：自己部署

如果你想自己部署到Cloudflare：

#### 第一步：准备工作

1. 注册Cloudflare账号（免费）
2. 安装Node.js（从 https://nodejs.org 下载）

#### 第二步：安装工具

打开命令行（Windows按Win+R，输入cmd），运行：

\`\`\`
npm install -g wrangler
\`\`\`

#### 第三步：登录Cloudflare

\`\`\`
wrangler login
\`\`\`

会自动打开浏览器，点击授权。

#### 第四步：创建存储桶

\`\`\`
wrangler r2 bucket create duxiu-pdf-storage
\`\`\`

#### 第五步：进入项目文件夹

\`\`\`
cd duxiu-zip2pdf-worker
\`\`\`

#### 第六步：安装依赖

\`\`\`
npm install
\`\`\`

#### 第七步：部署

\`\`\`
npm run deploy
\`\`\`

完成！会显示一个网址，比如：
\`\`\`
https://duxiu-zip2pdf-worker.你的名字.workers.dev
\`\`\`

打开这个网址就可以使用了！

## 💡 使用技巧

### 文件要求
- **格式**：ZIP、UVZ、CBZ
- **大小**：建议不超过100MB
- **内容**：必须包含图片文件

### 最佳实践
- 文件不要太大，建议50MB以内
- 如果图片很多（超过500张），可能需要等几分钟
- 网络不好时，可能需要重试

### 常见问题

**Q: 转换失败怎么办？**  
A: 检查ZIP文件是否损坏，是否包含图片

**Q: 要收费吗？**  
A: 个人使用完全免费（Cloudflare免费额度足够）

**Q: 文件存在云端安全吗？**  
A: 使用HTTPS加密传输，存储在Cloudflare的R2服务

**Q: 转换后的文件会保存多久？**  
A: 会一直保存，可以手动删除
